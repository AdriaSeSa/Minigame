# Minigame

## Description:
This project is based on the Stardew Valley's minigame Journey of the praire king.<br/>

## Key Features:
-Kill zombies<br/>
-Survive<br/>

## Control:
Enter: start game<br/>
R: Restart<br/>
Arrow keys: Move<br/>
Space: Attack<br/>
